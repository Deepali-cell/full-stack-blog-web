Blog Platform (Full-Stack)

This is a full-stack blog application where users can create, read, update, and delete articles. It also supports interactions like liking and commenting on blogs.
The project was built to understand modern full-stack patterns using Next.js (App Router), server actions, and schema validation.

Tech Stack
Frontend & Backend (Full-stack with Next.js)
Next.js (App Router)
Server Actions
Database & ORM
Prisma
(PostgreSQL or any DB used with Prisma)
Validation & UI
Zod (schema validation)
Tailwind CSS
ShadCN UI


User Features
User authentication (login/signup)
Create new blog posts
View blogs from other users
Like blogs
Comment on blogs
View personal dashboard

Dashboard Features
View all own articles
Add new articles
Edit existing articles
Delete articles

Core Functionality
Full CRUD operations for blog posts
Server Actions used for handling backend logic
Zod used for validating input data
Prisma for database operations
Clean UI using Tailwind + ShadCN
User-specific dashboard for content management

Architecture Overview
Next.js App Router handles both frontend and backend
Server Actions used instead of traditional REST APIs
Prisma used as ORM for structured database queries
Validation layer implemented using Zod schemas
Component-based UI structure

Key Learnings
Working with Next.js App Router and Server Actions
Using Prisma for database management
Implementing schema validation using Zod
Building full CRUD applications
Structuring a modern full-stack app without separate backend
Managing user-specific data (dashboard system)

Project Purpose
This project was built to understand modern full-stack development using Next.js, especially:
Server Actions vs traditional APIs
Data validation and type safety
Clean and scalable architecture
User-generated content systems
